BayesianOptimizationForNeuralNetwork
====

ニューラルネットワークのハイパーパラメーターの最適化をベイズ最適化で行いました。実装はChainer+GPyOptで行なっております。

bayesian_opt.pyを動かすと，ベイズ最適化を行います。
